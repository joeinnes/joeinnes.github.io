# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Personal portfolio index page for Joe Innes. Lists public projects with descriptions and links.

## Stack

Pure HTML and vanilla CSS — no build step, no dependencies, no package manager. Edit `index.html` directly.

## Adding a Project

To add a new project to the portfolio, follow the skill at `personal/.claude/skills/joeinnes-portfolio/SKILL.md`.

## Deployment

Deployed via GitHub Pages from the `main` branch. Pushing to `main` updates the live site.

## Code Style

- Single file: `index.html` contains all HTML, CSS, and any JS inline
- Raleway font from Bunny Fonts
- Dark theme with OKLCH colour variables
- Responsive: sidebar layout on desktop collapses to stacked on mobile
